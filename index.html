meshy

This is Meshy.

Motivation:

On more than one occasion, I've found myself in need of a simple (and free and accessible) tool for performing measurements, calculations, simple transformations, and other post-processing on digital meshes; the standard use case is 3D printing, e.g., jewelry. Existing tools are predominantly desktop-based, hence this project.

How-to:

A full user guide to Meshy in the repo.

Testing:

Some of my testing meshes.

Dependencies:

  • three.js
  • a slightly enhanced version of dat.gui (added functions for adding title text to controllers and folders, added a function to set display precision, and set the CSS to cut off overflowing text with an ellipsis)
  • Adam Hooper's js-priority-queue
  • Vadim Graboys's js-bintrees, specifically red-black trees
